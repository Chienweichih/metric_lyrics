# Metric Lyrics

[Metric][metric] discography, studio albums:

* Old World Underground, Where Are You Now? (2003)
* Live It Out (2005)
* Grow Up and Blow Away (2007)
* Fantasies (2009)
* Synthetica (2012)
* Pagans in Vegas (2015)
* Art of Doubt (2018)
* Formentera (2022)
* Formentera II (2023)

And single from `Scott Pilgrim vs. the World`

* Black Sheep (2010)

All the lyrics are organized in Markdown files, which can be converted into an EPUB using [Pandoc][pandoc], with the rules defined in the [Makefile](Makefile).

```sh
make list
```

```sh
make
```

```sh
make clean
```
